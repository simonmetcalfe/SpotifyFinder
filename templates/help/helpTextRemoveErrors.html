Refreshing a Playlist to Fix Remove Errors

Why do the remove errors occur

  1. Primary Reason: The playlist on the Spotify server needs to be refreshed.
    • Typically remove track errors only occur on playlist you created a long time ago.
    • Refreshing the playlist rewrites/updates the playlist track information.
    • Once the playlist is refreshed the remove errors will no longer occur.

  2. Secondary Reason: You change the same playlist in SpotifyFinder and Spotify at the same time.
    • Press 'Reload from Spotify' to update SpotifyFinder with the current track positons..

How to Refresh a Playlist

  • There are 2 ways to fix refresh a playlist.
    1. Use the Refresh Button on the Playlist Tab (described below).
    2. Manually Refresh the Playlist using the Spotify App (described below).
  • Either way will work. They both do the exact same thing. Pick one you are most comfortable with.

Refresh a Playlist using the Playlist Tab Refresh Button

  1. Go to the Playlist Tab.
  2. Select the playlist that has remove errors.
  3. Press the Refresh button.

How the Playlist Tab Refresh Button Works

  1. It verfies you own the playlist (you can not refresh a playlist you do not own).
  2. It makes a backup copy of the playlist.
    1. The backup playlist name will be: original playlist name appended with 'SF_Refresh_Backup_[date]_[time].
    2. The backup playlist will show up in your list of playlists.
    3. If the backup fails the refresh is terminated and the original playlist is not modified.
  3. It removes all the tracks from the playlist.
  4. It adds all the tracks back to the original playlist.
  5. Once you are satisfied with the refresh you can delete the backup playlist.
  6. Note: In the unlikely event that step 4 fails you will need to use the Spotify app to copy the tracks in the backup to the original playlist.
  • The above steps will preserve the original playlist id.
  • This is needed in cases where the playlist has been shared or is being followed.

Manually Refresh a Playlist Using the Spotify App - Overview

  1. Make a backup copy of the playlist that has remove errors.
  2. Remove from the tracks from the original playlist.
  3. Copy tracks from the backup to the original playlist.
  • The above steps will preserve the original playlist id.
  • This is needed in cases where the playlist has been shared or is being followed.

Manually Refresh a Playlist Using the Spotify App - Step by Step Instructions

  1. Open the Spotify Desktop App (not the Spotify Web App).
  2. Right click on the playlist that has remove errors.
  3. Click on 'Add to other playlist'.
  4. Click on 'New Playlist'.
  5. A new playlist will be created with all the tracks from the original playlist.
  6. The new playlist will be called: 'original playlist name (2)'.
  7. Go to original playlist.
  8. Click on first track.
  9. Press Ctrl-A or Cmd-A to select all the tracks.
  10. Press the Delete key and Press Remove. This will remove all the tracks from the original playlist.
  11. Right click on the backup playlist called: 'original playlist name (2).
  12. Click on 'Add to other playlist'.
  13. Select the original playlist. All the tracks from the backup will be added to the original playlist.
  14. Once you are satisfied the refresh is working you can delete the backup playlist.
  • The above steps will preserve the original playlist id.
  • This is needed in cases where the playlist has been shared or is being followed.
