Duplicates Tab

Overview

  • displays all duplicate tracks for the selected playlists
    • add/remove selected playlists on the Playlist Tab
  • remove a duplicate track from a specific playlist

Finding Dups (radio buttons)

  • there are 4 ways to search for duplicates:
    1. find dups across playlists using track id (exact match)
    2. find dups across playlists using "track name/artist name/track duration" (a very close match)
    3. find dups in the same playlist using track id (exact match)
    4. find dups in the same playlist using "track name/artist name/duration (a very close match)"

When finding dups using "track name/artist name/duration (a very close match)"

  • the track names will be identical (case-insensitive)
  • the artist names will be identical (case-insensitive)
  • the track duration will be within 10 seconds
  • the duration time difference between tracks can be changed using the dropdown list

Remove Selected Dups (button)

  • tracks with a checkmark present will be removed from the associated playlist
  • this feature uses both track id and track position to remove the track
  • up to 100 tracks can be selected
  • spotify limits the number of tracks that can be removed per call to 100
  • you can not remove tracks from a playlist you do not own
  • you can not remove tracks that have a null track id
  • if you get remove errrors have a look at the 'Refreshing a Playlist to Fix Remove Errors' Help page.

Remove Selected Dups Auto Select (drop down list)

  • this feature is designed to automate the selection of duplicate tracks
  • this feature is only enabled when 1 or 2 playlists are selected on the playlist tab
  • this feature will automatically select duplicate tracks (up to 100 tracks)
  • spotify limits the number of tracks that can be removed per call to 100
  • repeat the use of auto select until all the desired tracks are removed
  • if a column header was used to reorder the table, it will be returned to the original order by auto select
  • you can not remove tracks from a playlist you do not own
  • tracks that have a null track id are not auto selected

Remove Dups by Playlist (button) and Select Playlist (drop down list)

  • this feature is designed to automate the removal of All the duplicate tracks from the selected playlist
  • this feature is only enabled when 2 playlists are selected on the playlist tab
  • this feature is only enabled when finding dups across playlists
  • All the duplicate tracks for the selected playlist will be removed
  • if there are 1000 duplicated tracks listed for the selected playlist then all 1000 tracks will be removed
  • a dialog will be displayed showing the details prior to the remove, providing a chance to cancel the remove
  • this feature ignores/does not use the select checkboxes
  • this feature uses the track id to remove the tracks from the selected playlist
  • if the same track appears more than once in the selected playlist than all copies of the track will be removed
  • you can not remove tracks from a playlist you do not own
  • tracks that have a null track id are not removed

Why do tracks have the same name but different track ids

  • There are many reasons:
    1. The artist has made multiple recordings of the same track
    2. The artist releases the same track on multiple albums: studio, live, greatist hits, ...
    3. The artist re-releases the same album with improved sound (remastered)
    4. It is a long list....

Finding a Track

  • click in the search box at the bottom of TrackName column and then type the first few letters of the track name

Sorting (column headers)

  • you can click on any column header to sort the table
  • to return to the original sort order press the clear at the bottom of the page

Clear (button)

  • clears all selections and returns the table to the original sort order
  • refreshing your track table can be useful if you changed the column sort and want to revert to the original sorting
  • if you changed a playlist in the Spotify App: press 'Reload from Spotify' button to pull in the changes

Reload from Spotify (button)

  • reloads all of your playlists from Spotify
  • reloading your playlists can be useful if you change any of your playlists in the Spotify App while using this website

Null Track Id's

  • tracks with null track ids can not be removed
  • null track ids can occur if spotify lost the rights to the track or artist
